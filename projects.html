MS Word Clone (Single HTML File)

This project was a challenge to build a functional, browser-based document editor, mimicking key features of MS Word, all contained within a single HTML file without relying on external JavaScript libraries or complex frameworks. It demonstrates a deep understanding of core web technologies.

I implemented a responsive, ribbon-like interface with various formatting tools (bold, italic, underline, text alignment, font size, color). The project showcases effective use of internal CSS and Vanilla JavaScript for complex UI interactions and text manipulation, prioritizing performance and a lightweight footprint.
